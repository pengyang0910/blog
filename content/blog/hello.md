这是 Blog 分区的第一篇文章，用于展示博客的各种功能。

## 功能特性

现在博客已经具备了以下功能：

### 外观主题

- ✅ **深浅色主题切换**：点击右上角月亮/太阳图标
- ✅ **响应式设计**：完美支持移动端
- ✅ **自定义图标**：网站使用猫咪图标 🐱

### 内容组织

- ✅ **搜索功能**：快速找到你想要的内容
- ✅ **标签和分类**：更好地组织文章
- ✅ **归档页面**：按时间查看所有文章
- ✅ **面包屑导航**：清晰的页面路径

### 阅读体验

- ✅ **阅读时长**：自动计算阅读时间
- ✅ **字数统计**：显示文章字数
- ✅ **目录导航**：长文章自动生成目录（左侧边栏）
- ✅ **代码高亮**：支持多种编程语言
- ✅ **代码复制**：一键复制代码块

### 社交互动

- ✅ **文章分享**：分享到社交平台（可配置）
- ✅ **上下篇导航**：方便阅读系列文章
- ✅ **RSS 订阅**：订阅更新

### 侧边栏功能

- ✅ **左侧目录**：文章大纲快速跳转
- ✅ **右侧最新文章**：查看最近更新

## 代码示例

支持语法高亮的代码块。

### Python 示例

这是一个简单的 Python 函数：

```python
def hello_world():
    """
    打印欢迎信息
    """
    print("Hello, World!")
    return "欢迎来到我的博客"

if __name__ == "__main__":
    message = hello_world()
    print(f"消息: {message}")
```

### Go 语言示例

这是一个 Go 语言的例子：

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
    fmt.Println("欢迎来到我的博客")
}
```

### JavaScript 示例

前端开发常用的 JavaScript：

```javascript
// ES6 箭头函数
const greet = (name) => {
    console.log(`Hello, ${name}!`);
    return `欢迎 ${name} 来到我的博客`;
};

// 调用函数
greet("访客");
```
